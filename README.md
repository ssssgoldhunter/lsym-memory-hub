# lsym-memory

lsym 项目记忆体仓库 - 用于保存 AI 助手在 lsym 项目工作过程中的记忆、知识和文档。

## 📁 目录结构

```
lsym-memory/
├── project-overview/     # 项目概览
├── architecture/         # 架构设计文档
├── business-flows/       # 业务流程详解
├── technical-decisions/  # 技术决策记录
├── api-docs/            # API 接口文档
├── common-issues/        # 常见问题与解决方案
├── workflow/            # 工作流程与偏好
├── knowledge-base/       # 知识库
└── docs/                # 文档汇总
```
