## 一、充值退款变动明细 org_code 赋值修复

### 问题
`RefundRechargeTransAfter.java` 中变动明细的 `orgCode` 字段错误地从 `orgnTrans.getOperatorCode()` 取值，导致4张变动明细表（`trans_acct_change_detail_t`、`trans_acct_act_sum_change_detail_t`、`trans_acct_sum_change_detail_t`、`trans_sum_change_detail_t`）的 `org_code` 与交易主表 `trans_recharge_t` 不一致。

### 修复
- 文件：`RefundRechargeTransAfter.java` line 275
- 改动：`orgnTrans.getOperatorCode()` → `orgnTrans.getOrgCode()`
- 影响范围：4张变动明细表的 `org_code` 字段
- 推送分支：lsym_zyAccount_20260423、lsym_uat、lsym_test

## 三、平台收付款接口字段修正（核心改动）

### 问题
原来平台收款/付款的 `dealType`、`fundTp`、`outAcctNm`/`inAcctNm`、`bussId`、`bussSubId`、`payDate`、`payTime` 等字段放在 `reserve` map 中传递，导致银行端报参数校验失败（字段为空）。按照文档 4.6.14.6.2，这些字段应作为请求体直接字段。

### 涉及文件及改动

#### 1. ZxTransferRequest.java — 请求 DTO 新增字段
```java
// 新增
private String dealType;     // 交易类型 (00-分成服务费 01-违约金 02-罚款 03-其他)
private String contractId;   // 合同编号

// 字段重命名（对照文档域名）：
// userDNm  → outAcctNm  (付款用户名称/资金转出方，SM2加密)
// userCNm  → inAcctNm   (收款用户名称/资金转入方，SM2加密)
// transDt  → payDate     (实际交易日期，格式YYYYMMDD)
// transTm  → payTime     (实际交易时间，格式HHmmss)
```

#### 2. ZxTransTransferHandle.java — 业务代码修改

**platformPay（平台付款，bizFunc=2041）：**
```java
// 改动后：设为 ZxTransferRequest 直接字段
zxTransferRequest.setDealType(request.getDealType());
zxTransferRequest.setFundTp(request.getFundType());
zxTransferRequest.setInAcctNm(SMUtil.sm2EncryptHex(..., recBankAccountName));
zxTransferRequest.setBussId(request.getTransNo());
zxTransferRequest.setBussSubId(request.getTransSubNo());
zxTransferRequest.setPayDate(request.getTransDate());
zxTransferRequest.setPayTime(request.getTransTime());

// reserve 只保留 laasSsn
Map<String, Object> reserveMap = new HashMap<>();
reserveMap.put("laasSsn", zxTransferRequest.getLaasSsn());
```

**platformReceive（平台收款，bizFunc=2042）：**
- 用 `setOutAcctNm`（付款用户名称）而非 `setInAcctNm`
- `outAcctNo` = 付款方，`inAcctNo` = 平台账户
- memo 改为直接设 `zxTransferRequest.setMemo("平台收款")`

### 字段对照表（文档 4.6.14.6.2）

| 字段 | 域名 | 说明 | 付款必填 | 收款必填 |
|------|------|------|----------|----------|
| outAcctNm | outAcctNm | 付款用户名称(SM2加密) | 必填 | — |
| inAcctNm | inAcctNm | 收款用户名称(SM2加密) | — | 必填 |
| bussId | bussId | 商户业务订单号 | 必填 | 必填 |
| bussSubId | bussSubId | 商户业务子订单号 | 选填 | 选填 |
| payDate | payDate | 实际交易日期(YYYYMMDD) | 必填 | 必填 |
| payTime | payTime | 实际交易时间(HHmmss) | 必填 | 必填 |
| fundTp | fundTp | 资金类型 | 必填 | 必填 |
| contractId | contractId | 合同编号 | 选填 | 选填 |
